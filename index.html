CSS Properties

A few CSS properties commonly used to format and style text on web pages:

  • font-family — Sets the typeface used to display text. You can list multiple fonts as a fallback chain, such as Arial, Helvetica, or sans-serif.
  • color — Controls the foreground color of text. Accepts named colors, hex values, RGB, and more. This is my favorite property because it has the biggest visual impact with the least amount of code.
  • font-size — Defines the size of the text. Can be set in pixels, ems, rems, or percentages.
  • font-weight — Sets how thick or thin the text appears, such as normal, bold, or a numeric value like 600.
  • text-transform — Controls capitalization of text, with values like uppercase, lowercase, and capitalize.
